# customer-churn-analysis
Customer churn prediction with logistic regression in Python.

## About the project
This project predicts customer churn with logistic regression by using artificial data. The goal is to predict which customers are likely to churn based on behavioral and demographic indicators.

## Datasets
Source: Kaggle (https://www.kaggle.com/datasets/abdulwadood11220/netflix-customer-churn-dataset)
Files:
- historical_customers_data.csv
- current_customers_data.csv

## Methodology
- EDA and data preprocessing
- Building the logistic regression model
- Model evaluation
- Descriptive statistics

## Evaluation
The model has an accuracy of 0.89. This means that 89% of the customers in the test were classified correctly. The confusion matrix and classification report are showing balanced performance for both churn and non-churn classes.
